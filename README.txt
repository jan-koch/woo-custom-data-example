This plugin shows you how you can add custom meta fields to WooCommerce products.

== Description ==

A simple code example you can use to add your own custom fields to WooCommerce products.
It shows how to add custom meta fields in the WooCommerce "Product Data" area in the backend and how to access them in the frontend. 

== Installation ==

Download the plugin from the WordPress plugin repository and click "Install". You can also download the plugin manually from the WordPress repository, unzip it locally and upload the "woo-custom-data-example" to the wp-content/plugins folder on your server via FTP/SFTP.

== Frequently Asked Questions ==

= Does this plugin do anything by itself? =

No. It is meant to get you started with customizing WooCommerce products to your needs.

= Can I run it with other shop systems than WooCommerce? =

Sorry, this plugin is tailored to WooCommerce.
